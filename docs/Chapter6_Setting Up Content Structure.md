## 6.1. 添加内容类型
How to add and configure a new content type.
1. 管理菜单，结构-内容类型（admin/structure/types）。看到可用内容类型列表；
2. 点击增加内容类型。添加内容页面出现。填表。
3. 垂直tab中提交表格设置中，配置创建该页面内容需要的表格。
4. 垂直tab中发布选项中，决定这个新类型的内容的默认选项。填表。如果勾选Promoted to front page，则默认会显示到首页（不勾选）。修改设置不影响已经发布的条目。
5. 垂直tab中展示设置中，决定是否在内容条目中显示作者和发布日期。不勾选。
6. 垂直tab中菜单设置中，全部不选。
7. 点击保存和管理域。进入管理域页面，可以为内容类型添加域（6.3 为内容类型添加基本域）。
8. 重复上述过程，创建 食谱 内容类型。

## 6.2. 删除内容类型Deleting a Content Type
How to delete a content type that is not required for the site.
1. 管理菜单，结构-内容类型（admin/structure/types），看到内容类型列表。
2. 点击末尾下拉菜单中的删除；
3. 确认页中点击删除。
